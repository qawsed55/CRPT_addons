> ⚠️ **Внимание**: Это неофициальный набор утилит. Не имеет отношения к ООО «Оператор-ЦРПТ», системе «Честный ЗНАК» и ФНС России.

## 📦 Что внутри?

Коллекция вспомогательных скриптов и инструментов, облегчающих работу с веб-интерфейсом системы маркировки товаров. На данный момент включает:

- ✅ **Генератор XML** для УПД/СЧФДОП с поддержкой маркировки (КИЗ), таможенных данных и реквизитов из Excel
- ✅ **Экспорт в CSV** с гибкой настройкой формата (построчно / группировка)
- ✅ **Валидатор структуры** по актуальной XSD-схеме ФНС
